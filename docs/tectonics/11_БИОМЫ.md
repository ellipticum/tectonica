# 11. Биомная классификация

## Метод

Polygon lookup по диаграмме Уиттакера.
**Источник**: Whittaker 1975; Ricklefs & Relyea 2014; plotbiomes dataset.

Point-in-polygon тест (ray casting, Shimrat 1962) в координатах (T °C, P мм/год).

## Ecotone jitter

**Источник**: Risser 1995 (ecotone 10-50 км).

```
T_jitter = T + noise * 1.5°C
P_jitter = P + noise * 75 мм
```

## Полигоны Уиттакера

9 полигонов, проверяются по приоритету (первый match):

| Биом | ID | T диапазон | P диапазон |
|------|----|-----------|-----------|
| Tropical Rainforest | 6 | 20-30 | 2000-4500 |
| Subtropical Forest | 9 | 15-30 | 1200-4500 (L-shape) |
| Tropical Savanna | 7 | 20-30 | 500-2000 |
| Temperate Forest | 3 | 5-20 | 1000-4500 |
| Boreal Forest | 2 | -5-5 | 400-4500 |
| Mediterranean | 5 | 12-20 | 500-1200 |
| Temperate Grassland | 4 | 5-15 | 400-1200 |
| Steppe | 11 | 5-20 | 200-500 |
| Tundra | 1 | -15 — -5 | 0-4500 |

Default (no match): Desert (8).

## Тундра override (Köppen ET)

```
seasonal_amp = 20 * sin(lat)     — Terjung 1970
T_warmest = T + seasonal_amp / 2
if T_warmest < 10°C → Tundra
```

## Альпийская зона (ID 10)

**Источник**: Körner 2003, "Alpine Plant Life" Fig. 7.1.

```
treeline_base = max(4000 - 55 * |lat|, 200)
threshold = treeline_base + noise * 300
if h > threshold AND h > 500 → Alpine
```

## Riparian vegetation

Реки > 0.12 интенсивности улучшают сухие биомы:
- T > 20°C → Tropical Savanna
- T > 12°C → Subtropical Forest
- else → Temperate Forest

## Mode filter

2 прохода: если < 2 из 4 cardinal neighbors совпадают — замена.
