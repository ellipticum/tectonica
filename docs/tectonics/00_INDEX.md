# Tectonics Blueprint v2

## Зачем этот модуль

Этот модуль фиксирует научно-обоснованную модель тектоники, чтобы:

1. Убрать «нарезанные ломти» и геометрически примитивные плиты.
2. Перейти от декоративных обводок к причинной геодинамике.
3. Связать форму континентов, гор и океанического рельефа через одну физическую систему.
4. **Обеспечить единый пайплайн для всех scope: planet, island, continent chunk.**

## Инварианты модели

1. Плита — это динамический объект на сфере, а не статичный полигон.
2. Тип границы определяется относительной кинематикой, а не только геометрией.
3. Океанический рельеф определяется возрастом коры, тектоникой и осадками.
4. Горные системы формируются в полях деформации (широких зонах), а не по «тонкой линии».
5. Все рендер-слои строятся из физических полей, а не наоборот.
6. **Один пайплайн, два режима координат.** Planet scope и island scope используют одну физику.
7. **Высота определяется изостатикой.** `h = f(crust_thickness)`, не эмпирическими формулами.
8. **Рельеф острова — баланс подъёма и эрозии.** `dh/dt = U - K*A^m*S^n + kappa*nabla^2(h)`.
9. **Береговая линия — не рисуется, а возникает** из пересечения рельефа и уровня моря.
