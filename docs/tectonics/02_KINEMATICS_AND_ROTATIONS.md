# 02. Кинематика На Сфере И Полюса Эйлера

## 1. Базовая кинематика

Для жесткой плиты движение на сфере задается вектором угловой скорости `Ω` (полюс Эйлера + скорость вращения).

Формула скорости точки на сфере:

`v(r) = Ω × r`

где:

1. `r` — радиус-вектор точки на сфере.
2. `v` — локальная линейная скорость.

Эта схема является стандартом в реконструкциях плит и реализована в GPlates/pyGPlates [R3], [R5].

## 2. Относительная скорость на границе

Для соседних плит `A` и `B`:

`v_rel = v_A - v_B = (Ω_A - Ω_B) × r`

Далее раскладываем `v_rel` на нормальную и тангенциальную компоненты к локальной линии границы:

1. `v_n = dot(v_rel, n)` — сходимость (`v_n < 0`) или расходимость (`v_n > 0`).
2. `v_t = dot(v_rel, t)` — сдвиг (transform компонент).

## 3. Правило классификации границ

Пороговые критерии:

1. `v_n > +eps_div` и `|v_t| < eps_shear` => дивергентная.
2. `v_n < -eps_conv` и `|v_t| < eps_shear` => конвергентная.
3. `|v_n| <= eps_mix` и `|v_t| >= eps_shear` => трансформная.
4. Иначе => смешанный/диффузный режим.

Рекомендуемые стартовые пороги:

1. `eps_div = 2 mm/yr`
2. `eps_conv = 2 mm/yr`
3. `eps_shear = 4 mm/yr`
4. `eps_mix = 1 mm/yr`

## 4. Временная эволюция плит

Для шага `dt`:

1. Обновить `Ω_i(t)` для каждой плиты.
2. Адвектировать вершины полигонов по `v(r)`.
3. Перестроить `BoundaryGraph`.
4. Пересчитать `v_n`, `v_t`, типы границ и ширины деформационных коридоров.

## 5. Тройные узлы и устойчивость

В узле из трех границ должны выполняться кинематические ограничения совместимости скоростей.  
На практике:

1. Проверять невязку по замыканию скоростей.
2. При большой невязке инициировать событие реорганизации узла.

Это критично, иначе появляются неустойчивые «ломаные» структуры и негеологичные линии.

## 6. Почему это важно для формы континентов

Если контуры движутся как динамические объекты в поле `Ω`, форма суши становится исторически обусловленной:

1. Появляется направленная вытяжка и изгиб по движению.
2. Границы перестают быть геометрическими «разрезами».
3. Возникают цепочки дуг/бассейнов/террейнов, похожие на реальные.
