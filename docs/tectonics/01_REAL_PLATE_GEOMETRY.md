# 01. Реальная Геометрия Плит

## 1. Главный вывод

Да, текущая «ломтиковая» картинка — признак упрощенной схемы.  
Реальные плиты не выглядят как чистый Voronoi:

1. Границы не прямые и не равномерные по кривизне.
2. Есть микроплиты и блоки.
3. Есть диффузные деформационные пояса (широкие зоны, не одна линия).
4. Границы мигрируют, раскалываются и переорганизуются во времени.

## 2. Что говорит литература

1. Глобальная цифровая модель границ плит (PB2002) учитывает не только «основные» плиты, но и орогенические/деформационные типы границ, что уже отражает сложность геометрии [R1].
2. MORVEL56 показывает 56 плит и отдельное рассмотрение диффузных границ, то есть чисто «жесткий мозаичный» мир недостаточен [R2].
3. Современные реконструкции используют топологически замкнутые полигоны и их эволюцию по времени, а не фиксированные контуры [R3], [R4].

## 3. Почему Voronoi дает «неприродный» результат

1. Статичные центры генерации создают полигоны с близкой характерной шкалой.
2. Граница определяется метрикой ближайшего центра, а не историей деформации.
3. Нет механизма естественного ветвления границ и «затухания» в диффузных зонах.
4. Нет временной памяти: прошлые события не формируют сложную морфологию.

## 4. Требования к новой геометрии плит

## 4.1 Представление

1. `PlatePolygon(t)` — замкнутые топологические полигоны на сфере.
2. `BoundaryGraph(t)` — граф границ между соседними плитами.
3. `DeformationCorridors(t)` — поле ширины/интенсивности деформации вблизи границ.
4. `MicroplateCandidates(t)` — локальные блоки, которые могут отделяться.

## 4.2 Эволюция

1. Адвекция контуров по полю скоростей.
2. События split/merge/transfer в тройных узлах.
3. Рекомпоновка границ при смене относительных движений.

## 4.3 Ограничения

1. Полигоны должны оставаться без самопересечений.
2. Планета покрыта плитами полностью, без дыр и перекрытий.
3. Суммарная длина и типы границ должны статистически попадать в земной диапазон.

## 5. Минимальная «землеподобная» цель

Чтобы убрать «ломтики», нужно выполнить три условия одновременно:

1. Неравномерные размеры плит с длинным хвостом (несколько больших + множество средних/малых).
2. Гибрид границ: резкие + диффузные.
3. Длительная эволюция (сотни шагов), чтобы формы «созрели», а не были сгенерированы мгновенно.
