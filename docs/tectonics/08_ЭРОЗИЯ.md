# 08. Эрозия и формирование рельефа

## Stream power эрозия

**Источник**: Braun & Willett 2013, O(n) implicit solver.

```
E = K_eff * A^m * S^n
h_new = (h_old + U*dt + factor * h_recv) / (1 + factor)
factor = K_eff * dt * A^m / dx^n
```

### Sub-grid channel width scaling

**Источник**: Pelletier 2010; Leopold & Maddock 1953.

```
Q = A * runoff                  — расход воды
W = k_w * Q^0.5                — ширина канала (Leopold & Maddock 1953)
K_eff = K_rock * (W / dx)      — масштабирование на долю ячейки
```

| Параметр | Значение | Источник |
|----------|----------|----------|
| k_w | 0.005 m^0.5 s^-0.5 | Leopold & Maddock 1953 |
| runoff | 0.5 м/год | Fekete et al. 2002 (global mean) |

Предотвращает чрезмерную эрозию 10 км ячеек.

### Стохастическое направление потока

**Источник**: Tucker & Bras 2000.

В D8 flow routing к уклону добавляется шум ±5%:
```
slope_perturbed = slope + noise * 0.05 * |slope|
```

Ломает grid-alignment bias at source (физическая мотивация:
турбулентная вариативность потока на sub-grid масштабах).
Заменяет предыдущий хак noise injection ±5 м.

### MFD дренажная площадь

**Источник**: Freeman 1991; goSPL (Salles et al., Science 2023).

```
w_j = max(0, slope_j)^p
area[j] += area[i] * w_j / sum(w)
```

## Uplift rates

**Источник**: GPS-данные (Bevis et al. 2005; Calais et al. 2003).

| Тип | Формула | Целевой диапазон |
|-----|---------|-----------------|
| Конвергент | bstr * 0.008 * speed_factor | 5-10 мм/год |
| Дивергент | -bstr * 0.002 * speed_factor | -1-3 мм/год |
| Трансформ | bstr * 0.0005 * speed_factor | <1 мм/год |

## Изостатическая релаксация

**Источник**: Watts 2001 §8.4.

```
tau_eff = 5 Myr
f_relax = 1 - exp(-1.5/5.0) ≈ 0.26
h = h_eroded * 0.74 + h_isostatic * 0.26
```

## Остаточная гипсометрическая коррекция

**Источник**: Harrison et al. 1983; Cogley 1984.

С water loading + thermal subsidence — только для residual эффектов
(седиментная нагрузка, динамическая топография).

Trigger: median freeboard > target * **1.5** (ранее 1.2).

## Detail noise

**Источник**: Huang & Turcotte 1989 (beta=2.0); Montgomery & Brandon 2002.

4 октавы, спектральный наклон beta=2.0:

| Freq | Amplitude | Lambda |
|------|-----------|--------|
| 16 | 80 м | ~400 км |
| 32 | 40 м | ~200 км |
| 64 | 20 м | ~100 км |
| 128 | 10 м | ~50 км |

Elevation scaling: `sqrt(h / 5000)`, clamp [0, 1].
