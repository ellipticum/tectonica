# 06. Изостазия и батиметрия

## Airy isostasy с water loading

**Источник**: Turcotte & Schubert 2002, §2.2, §2.6.

Изостатический баланс с учётом массы воды в океанических бассейнах:

```
Конт.:  h = C * (rho_m - rho_c_eff) / rho_m
Океан:  h = C * (rho_m - rho_c_eff) / (rho_m - rho_w)
Blend:  denom = rho_m - (1 - continental_frac) * rho_w
```

Water loading (rho_w = 1025 кг/м3) увеличивает глубину океана
на множитель rho_m / (rho_m - rho_w) = 1.45.

## Плотности

| Параметр | Значение | Источник |
|----------|----------|----------|
| rho_c (конт.) | 2800 кг/м3 | Стандартное |
| rho_c (океан.) | 2900 кг/м3 | Стандартное |
| rho_m (мантия) | 3300 кг/м3 | Стандартное |
| rho_w (вода) | 1025 кг/м3 | Turcotte & Schubert §2.6 |

Интерполяция: `rho_c = 2900 - continental_frac * 100`.

## Термальная коррекция

```
rho_c_eff = rho_c * (1 - heat_anomaly * 0.02)
```

- alpha = 3e-5 /K (Turcotte & Schubert 2002 §4.3)
- 2% верхняя граница (включает partial melt)

## Thermal subsidence

**Источник**: Parsons & Sclater 1977; Stein & Stein 1992 (GDH1).

Дифференциальная субсидия от хребта:
```
t < 80 Ma:  delta_d = 350 * sqrt(t_Ma)    — half-space cooling
t >= 80 Ma: delta_d = 3151 - 2473 * exp(-0.0278 * t)  — plate model
```

Возраст вычисляется из BFS-расстояния до дивергентных границ:
```
age_Ma = dist_km / (v_half_cm_yr * 10)
```

## Флексуральная изостазия

**Источник**: Watts 2001; Turcotte & Schubert 2002 §3.13.

Гауссово сглаживание с физически обоснованным числом проходов:
```
D = E * Te^3 / [12(1 - nu^2)]     — жёсткость плиты
alpha = [4D / (delta_rho * g)]^(1/4) — флексуральный параметр
N = alpha^2 / (2 * dx^2)           — число проходов
```

| Параметр | Значение | Источник |
|----------|----------|----------|
| E | 70 GPa | Среднее |
| nu | 0.25 | Poisson |
| Te | 25 км | Watts 2001 Table 5.1 |
| delta_rho | 900 кг/м3 | rho_m - rho_infill |

→ D ≈ 9.7e22 N·m, alpha ≈ 83 км, **N ≈ 34 прохода**.
