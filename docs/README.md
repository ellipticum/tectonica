# Project Documentation

Эта директория хранит инженерные спецификации по генератору планеты.

## Структура

- `docs/relief` — текущий blueprint по рельефу и смежным слоям.
- `docs/tectonics` — новый полный блок по физике и математике тектоники плит.

## Принцип

Код меняется только после фиксации модели в документации:

1. Физические допущения.
2. Уравнения и параметры.
3. Численная схема и ограничения.
4. Метрики валидации.
